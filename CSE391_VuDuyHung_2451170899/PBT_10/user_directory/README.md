# 👥 User Directory
Ứng dụng quản lý người dùng với chức năng CRUD.

### 🚀 API đã dùng:
- [JSONPlaceholder API](https://jsonplaceholder.typicode.com/): Endpoint `/users` để giả lập thao tác quản lý dữ liệu.

### 🛠️ Cách chạy:
1. Mở file `index.html` bằng trình duyệt web.
2. Đảm bảo kết nối mạng để API phản hồi.
3. Khuyên dùng **Live Server** (VS Code).
