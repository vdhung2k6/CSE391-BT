# 🌤️ Weather App
Ứng dụng tra cứu thời tiết thời gian thực.

### 🚀 API đã dùng:
- [Open-Meteo Geocoding API](https://geocoding-api.open-meteo.com/): Tìm kiếm tọa độ dựa trên tên thành phố.
- [Open-Meteo Forecast API](https://api.open-meteo.com/): Lấy dữ liệu thời tiết (nhiệt độ, gió, code thời tiết).

### 🛠️ Cách chạy:
1. Mở file `index.html` bằng trình duyệt web.
2. Khuyên dùng extension **Live Server** (VS Code) để chạy dự án.
