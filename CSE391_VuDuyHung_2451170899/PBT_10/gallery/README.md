# 🖼️ Infinite Scroll Gallery
Thư viện ảnh cuộn vô hạn.

### 🚀 API đã dùng:
- [Picsum Photos API](https://picsum.photos/): Endpoint `/v2/list` lấy danh sách ảnh ngẫu nhiên.

### 🛠️ Cách chạy:
1. Mở file `index.html` bằng trình duyệt.
2. Cuộn chuột xuống dưới để ứng dụng tự động tải thêm ảnh (Infinite Scroll).
